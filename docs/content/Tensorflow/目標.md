> appleFace.json，appleLeftEye.json，appleRightEye.json

|  |  |
| :-: | :-: |
| H | 邊界框的高度（以像素為單位） |
| W | 邊界框的寬度（以像素為單位） |
| X, Y | 邊界框左上角的位置（以像素為單位） |
| IsValid | 是否真的有檢測.1 =檢測; 0 =沒有檢測。 |
