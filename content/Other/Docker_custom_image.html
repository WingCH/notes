如果每次都要自己安裝十分麻煩，所以嘗試自己包裝一個image

建立一個Dockerfile檔案

FROM alpine

RUN apk update && apk upgrade #可忽略
RUN apk add vim

利用Dockerfile生成一個image

docker build -t alpine-with-vim .
